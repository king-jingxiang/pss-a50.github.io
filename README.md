# PSS-A50 音色库展示页面

一个美观、交互式的Yamaha PSS-A50电子琴音色库展示页面，支持在线试听所有音色。

## 🎹 项目特色

- **完整的音色展示**：展示PSS-A50电子琴的全部42种音色
- **在线试听功能**：每个音色都支持在线试听
- **智能搜索**：支持按音色名称、中文名、描述、分类搜索
- **分类筛选**：12个音色分类，快速找到所需音色
- **响应式设计**：完美适配移动端和桌面端
- **现代化界面**：渐变背景、动画效果、卡片式布局

## 🎵 音色分类

- **钢琴 (Piano)** - 4种音色
- **风琴 (Organ)** - 4种音色  
- **吉他 (Guitar)** - 5种音色
- **贝斯 (Bass)** - 4种音色
- **弦乐 (Strings)** - 5种音色
- **木管 (Woodwind)** - 4种音色
- **铜管 (Brass)** - 3种音色
- **合成音色 (Synth)** - 4种音色
- **合成Pad (Synth PAD)** - 3种音色
- **管弦/其他** - 2种音色
- **打击乐 (Drum Kit / Percussion)** - 4种音色

## 📱 功能预览

### 搜索功能
- 支持中英文搜索
- 实时筛选显示结果
- 支持按音色特点搜索

### 分类筛选
- 一键筛选特定分类
- 显示当前筛选统计
- 快速切换不同分类

### 音频播放
- 点击试听按钮播放音色
- 播放状态视觉反馈
- 支持播放/暂停切换
- 自动停止其他正在播放的音色

## 🛠️ 本地运行

### 方法一：使用Python HTTP服务器
```bash
# 在项目根目录运行
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

### 方法二：使用Node.js
```bash
# 安装依赖
npm install
# 启动开发服务器
npm start
# 访问 http://localhost:8000
```
